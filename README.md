# CLI-Game
1. Класс Game - основной игровой цикл
Назначение: Реализация основного игрового цикла и координация всех компонентов игры.

Ключевые функции:

runGame() - главный игровой цикл

playerTurn() - обработка хода игрока

castSpell() - применение заклинаний

Обработка команд пользователя

Игровой цикл включает:

Начало игры и инициализация

Запуск уровня через LevelManager

Ход игрока (движение, атака, заклинания)

Ход врагов через CombatManager

Ход башен через CombatManager

Проверка условий завершения уровня

Обработка победы/поражения

2. Класс GameState - управление состоянием игры
Назначение: Централизованное хранение и управление всем состоянием игры.

Содержит:

Игровое поле (GameField)

Объект игрока (Player)

Коллекцию игровых объектов

Текущий статус игры

Уровень сложности

Ключевые методы:

Управление жизненным циклом игровых объектов

Фильтрация объектов по типам (враги, башни, ловушки)

Обновление визуального состояния поля

3. Класс LevelManager - система уровней
Назначение: Управление переходами между уровнями и их инициализацией.

Ключевые функции:

startLevel() - запуск нового уровня

nextLevel() - переход на следующий уровень

Инициализация врагов и башен

Особенности реализации:

Увеличение размера поля: 15 + (уровень-1)*2

Рост количества врагов: 5 + уровень

Рост количества башен: 2 + уровень/2

Случайная генерация позиций с проверкой валидности

4. Класс CombatManager - система боя
Назначение: Обработка боевых взаимодействий и AI врагов.

Ключевые функции:

performRangedAttack() - дальняя атака игрока

processEnemyTurns() - ход врагов и башен

movePlayer() - движение и ближний бой

Особенности AI:

Враги двигаются случайным образом

Башни атакуют игрока в радиусе действия

Автоматическая атака при столкновении

5. Класс SaveLoadManager - система сохранения
Назначение: Сохранение и загрузка игры с обработкой исключительных ситуаций.

Ключевые особенности:

Использование RAII для управления файлами

Сериализация всех компонентов игры

Детальная обработка ошибок

Сохраняемые данные:

Текущий уровень и размеры поля

Состояние игрока и его заклинаний

Позиции и состояние всех игровых объектов

Текущий статус игры

6. Иерархия исключений
Базовый класс: GameException

SaveGameException - ошибки сохранения

LoadGameException - ошибки загрузки

FileNotFoundException - файл не найден

CorruptedSaveException - поврежденные данные

Все исключения содержат подробные сообщения об ошибках и обрабатываются без аварийного завершения программы.
